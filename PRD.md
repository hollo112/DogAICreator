# PRD: DogAICreator - 강아지 AI 영상 생성기

## 1. 개요

강아지 사진을 업로드하고 프롬프트를 작성하면, Gemini API를 활용하여 강아지가 말하거나 행동하는 AI 영상을 생성하는 웹 애플리케이션.

## 2. 목표

- 유튜브에서 유행하는 "말하는 강아지" AI 영상을 누구나 쉽게 만들 수 있도록 한다
- 모바일/PC 모두에서 편리하게 사용할 수 있어야 한다
- 생성된 영상을 바로 다운로드할 수 있어야 한다

## 3. 기술 스택

- **Frontend/Backend**: Streamlit (Python)
- **AI API**: Google Gemini API (영상 생성)
- **언어**: Python 3.14+
- **배포**: Streamlit Cloud 또는 로컬 실행

## 4. 핵심 기능

### 4.1 강아지 사진 업로드
- 사용자가 강아지 사진을 업로드할 수 있다
- 지원 포맷: JPG, PNG, WEBP
- 모바일에서도 카메라 촬영 또는 갤러리에서 선택하여 업로드 가능
- 업로드된 사진 미리보기 제공

### 4.2 프롬프트 입력
- 사용자가 원하는 영상 내용을 텍스트로 입력
- 예시: "안녕하세요 저는 멍멍이입니다. 오늘 산책 가고 싶어요!"
- 프롬프트 예시를 보여줘서 사용자가 참고할 수 있도록 한다
- 한국어/영어 프롬프트 모두 지원

### 4.3 AI 영상 생성
- Gemini API를 호출하여 강아지 사진 + 프롬프트 기반 영상 생성
- 강아지가 말하는 것처럼 입이 움직이는 영상
- 프롬프트에 대사를 포함하면 TTS(음성 합성)로 음성도 추가
- 생성 중 로딩 상태 표시 (프로그레스 바 또는 스피너)

### 4.4 영상 미리보기 및 다운로드
- 생성된 영상을 웹 페이지에서 바로 재생
- 다운로드 버튼을 통해 MP4 파일로 다운로드
- 영상 생성 이력 관리 (세션 내)

## 5. UI/UX 요구사항

### 5.1 레이아웃
- 심플하고 직관적인 단일 페이지 구성
- 상단: 앱 타이틀 및 설명
- 중단: 사진 업로드 영역 + 프롬프트 입력 영역
- 하단: 생성 버튼 → 결과 영상 표시 + 다운로드 버튼

### 5.2 반응형 디자인
- 모바일 화면에서도 정상적으로 동작
- 터치 친화적 UI
- Streamlit의 기본 반응형 레이아웃 활용

### 5.3 사용자 경험
- 사진 업로드 후 썸네일 미리보기
- 프롬프트 예시 버튼 (클릭하면 자동 입력)
- 영상 생성 중 예상 대기 시간 안내
- 에러 발생 시 친절한 안내 메시지

## 6. API 연동

### 6.1 Gemini API
- Google Gemini API를 사용하여 영상 생성
- API 키는 환경변수 또는 Streamlit secrets로 관리
- API 호출 실패 시 재시도 로직 포함
- 요청 제한(Rate Limit) 처리

## 7. 비기능 요구사항

- 업로드 파일 크기 제한: 최대 10MB
- 영상 생성 타임아웃: 최대 5분
- API 키는 코드에 하드코딩하지 않는다
- .env 또는 Streamlit secrets 사용

## 8. 향후 확장 가능성 (v2)

- 다양한 동물 지원 (고양이 등)
- 영상 스타일 선택 (재미있는, 귀여운, 진지한)
- SNS 공유 기능
- 영상 길이 조절
- 배경 음악 추가
